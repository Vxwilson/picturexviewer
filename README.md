# picturexviewer
A simple Python image viewer with side-by-side slideshow capability.

# Usage
Select image files by pressing the hotkey ``` <Ctrl+O> ```. To start a slideshow, press ``` <S> ``` and select options such as ``` side images count ```.

# Prerequisite
None.
